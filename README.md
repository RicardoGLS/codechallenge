# Codechallenge (Frontend - SimulateBack)
 Codechallenge Frontend - Angular - Nodejs



# Descripción

# Requerimientos
- Angular cli
- Nodejs


# Instalación de librerias
```
npm install
```




# Inicialización local
En la carpeta ejecutar los siguientes comandos:
```
npm run start
npm run server
```
